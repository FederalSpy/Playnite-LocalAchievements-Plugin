# Security Policy / Política de Seguridad

## English
This is a hobby project.

If you find a security issue, please open a private report if possible or contact the maintainer before public disclosure.

Include:
- Steps to reproduce
- Affected versions
- Potential impact

## Español
Este es un proyecto hobby.

Si encuentras un problema de seguridad, abre un reporte privado si es posible o contacta al mantenedor antes de divulgarlo públicamente.

Incluye:
- Pasos para reproducir
- Versiones afectadas
- Impacto potencial
